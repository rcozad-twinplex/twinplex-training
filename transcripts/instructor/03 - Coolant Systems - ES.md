**Objetivo n. ° 2: Nivel del depósito de refrigerante**

Dada cualquier presión, verifique, mezcle y vuelva a llenar el depósito según los criterios que se encuentran en el Control de rendimiento, de modo que el nivel del depósito se mantenga constantemente.

## ¿Qué es un sistema de refrigeración?

*Un* sistema de refrigeración es el sistema que entrega el compuesto de estirado al área del punzón y la matriz de la prensa. Este compuesto de trefilado enfría y lubrica a medida que se forma metal en la prensa. Para producir piezas de calidad, es esencial que este *"refrigerante"* se mezcle de manera constante y se entregue al área de herramientas. Es su responsabilidad asegurarse de que esto suceda.

## ¿Cómo funciona un sistema de refrigeración?

*Un* sistema de refrigeración toma el compuesto de extracción almacenado en un depósito y lo bombea a través de varias mangueras y líneas. Las líneas de refrigerante se pueden ajustar para que *"la cantidad justa"* de compuesto de dibujo se coloque en *"el lugar correcto"* durante todo el proceso de estampado.

Un sistema de refrigeración se compone de varios componentes. Incluyen:

- El depósito, que almacena el refrigerante.
- La bomba, que mueve el refrigerante.
- Las mangueras, que llevan el refrigerante a la barra de refrigerante.
- El filtro, que filtra los contaminantes sólidos.
- La barra de refrigerante, en la que se fijan las líneas ajustables.
- Las líneas de refrigerante, que dirigen y ajustan la tasa de flujo de refrigerante a las herramientas.

## ¿Qué tipo de aceite se utiliza en un sistema de refrigeración?

El *"refrigerante"* utilizado tiene propiedades tanto de refrigeración como de lubricación. Por esa razón, se describe con mayor precisión como un *"compuesto de dibujo"*. Oirá que ambos términos se usan indistintamente a veces. Para evitar confusiones, usaremos la palabra *"refrigerante"* para referirnos a cualquier aceite o compuesto de dibujo utilizado para enfriar y lubricar las herramientas en el proceso de estampado.

El refrigerante que usamos puede ser diferente para cada trabajo que realizamos. El tipo utilizado para cada trabajo a veces se enumera en varios lugares, como en la hoja de proceso o en una etiqueta magnética pegada a la prensa.

Algunos de los refrigerantes se denominan *"solubles en agua"*. Esto significa que el aceite debe mezclarse con agua. Dos cosas importantes para recordar son:

- **La proporción de agua a aceite:** la proporción es extremadamente importante. Por ejemplo, siempre mezcle el refrigerante en la proporción especificada para el trabajo. Una proporción de 10: me refiero a diez partes de agua por una parte de aceite.
- **El método de mezcla.** Siempre mezcle el aceite con el agua, no al revés. Si lo mezcla incorrectamente, el refrigerante podría perder sus propiedades de lubricación. Esto podría causar rayaduras o rozaduras graves en las piezas.

## Principios del sistema de refrigerante

Debe recordar algunos principios clave sobre los sistemas de refrigeración para realizar su trabajo de manera eficaz. Incluyen:

- **Conozca su sistema de refrigeración.**

  Tenemos una variedad de prensas y, por esa razón, también una variedad de sistemas de refrigeración. Si bien realizan básicamente la misma función, es importante saber lo siguiente para cada sistema de refrigeración:

  Dónde se encuentra el depósito.

  Qué refrigerante se utiliza para el trabajo actualmente en la prensa.

  El caudal de refrigerante para el trabajo actual.

  Los procedimientos de ubicación y mantenimiento del filtro.

  Cualquier instrucción especial para cada sistema de refrigeración.

- **Utilice el refrigerante correcto.**

  Se especifica el tipo de refrigerante utilizado para cualquier trabajo. Consulte a su supervisor para determinar qué tipo de refrigerante utilizar para cada trabajo y dónde se puede encontrar esta información. Si no lo sabe, pregunte.

- **Compruebe el nivel del depósito con frecuencia.**

  El refrigerante normalmente se inunda en las herramientas durante el funcionamiento. El refrigerante lo llevan a cabo las piezas producidas y la chatarra generada. Por esta razón, siempre verifique el nivel de refrigerante con • frecuencia tanto como cada vez que se acerque a la prensa.

- **Compruebe el caudal de refrigerante.**

  A veces las cosas se pueden conseguir en el refrigerante Lin es y taparlas. Verifique el caudal de cada línea de refrigerante cada vez que se acerque a la prensa para asegurarse de que permanezca constante.

- **Mezcle el refrigerante correctamente.**

  Hace una gran diferencia cómo se mezclan algunos aceites. Algunos de los refrigerantes que usamos son solubles en agua, es decir, debe mezclarlos con agua. Para gestionar este proceso de mezcla para que nunca

- **Mire sus proporciones de mezcla.**

  La proporción de mezcla es la cantidad de agua a la cantidad de aceite. Una proporción de 2: 1 sería dos partes de agua y una parte de aceite. Una proporción de 1 O: 1 sería diez partes de agua y una parte de aceite. Mezcle siempre en la proporción especificada. Use un refractómetro para probar sus proporciones de mezcla.

- **Esté atento a la evaporación del agua.**

  Ou r puestos de trabajo a largo corriendo el agua en la mezcla refrigerante puede evaporarse. Esto conduce a una mezcla espesa. Para corregir esto, mezcle un lote de refrigerante que tenga una proporción mucho mayor de agua. Revuelva esta mezcla *"fina"* en el tanque de refrigerante. Nunca mezcle con agua corriente, el refrigerante podría perder sus propiedades de lubricación.

- **Vertido de refrigerante.**

  Cuando se extrae refrigerante de una prensa, se debe desechar correctamente. Pregúntele a su supervisor dónde se desecha el refrigerante viejo. Nunca lo deseche por un desagüe estándar. El vertido adecuado de refrigerante viejo podría provocar problemas medioambientales y multas importantes.
