## ¿Qué es un sistema de lubricación?

El sistema de lubricación de una prensa es como el aceite de motor de un automóvil. Proporciona una lubricación fundamental a los cojinetes, las piezas móviles y las superficies de desgaste.

Todos sabemos que el nivel de aceite en el motor de un automóvil debe controlarse y mantenerse constantemente. El resultado de no monitorear este nivel lo suficientemente cerca son miles de dólares en facturas de reparación y muchos dolores de cabeza. El sistema de lubricación de la prensa es idéntico en el sentido de que debe controlarse constantemente. Esto asegura que el sistema lubricará de manera efectiva los cojinetes y las superficies críticas.

## ¿Cómo funciona un sistema de lubricación?

Un sistema de lubricación tiene cinco componentes principales:

- Aceite, que proporciona lubricación.
- Depósito, que almacena el aceite.
- Bomba, que mueve el aceite.
- Filtro, que filtra las impurezas sólidas del aceite.
- Tubería, que distribuye aceite a cada ubicación.
- Accesorios, que controlan el flujo de aceite en cada ubicación.

El aceite utilizado en cada prensa es un tipo específico recomendado por el fabricante de la prensa. Este aceite se almacena en un depósito, se bombea a través de un tubo a cada ubicación crítica y se fuerza a través de un accesorio especial que controla la cantidad de aceite que se distribuye en esa ubicación.

## Principios de lubricación

Debe recordar algunos elementos clave sobre la lubricación para realizar su trabajo de manera efectiva. Éstos incluyen:

**Conozca su sistema de lubricación.**

Tenemos una variedad de prensas y, por esa razón, también una variedad de sistemas de lubricación. Si bien todos realizan básicamente la misma función, es importante saber lo siguiente para cada sistema de lubricación de prensa:

- Dónde se encuentra el depósito
- Qué aceite se utiliza.
- Qué presión de aceite se requiere.
- Cualquier instrucción especial para cada sistema.

- **El tipo de aceite varía según la prensa; utilice el aceite correcto.**
- No todas las prensas utilizan el mismo aceite. El fabricante de la prensa recomienda qué tipo de aceite utilizar según los requisitos de la prensa. Algunos aceites proporcionan más lubricidad, otra más resistencia al desgaste, etc. Es de vital importancia utilizar el aceite recomendado para la prensa.

- **Compruebe el nivel del depósito con frecuencia.**
- El sistema de lubricación de **una** prensa está diseñado para utilizar aceite. Por esta razón, el depósito debe revisarse con frecuencia. Si el nivel del depósito permanece constante, podría indicar que no llega aceite a áreas críticas.

- **Controle la presión del aceite.**
- La presión en el sistema de lubricación debe permanecer constante entre 20 y 40 psi. Si cae fuera de este rango, dígaselo a su líder de equipo o supervisor.

- **Supervisar el sistema para prevenir p problemas usa el aceite correcto.**
  - Mantenga el nivel de aceite en el depósito.
  - Controle la presión del aceite. Debe permanecer entre 20 y 40 psi.
  - Si el nivel del depósito permanece igual durante períodos prolongados mientras la prensa está funcionando, podría indicar que algo no funciona como debería.
  - Si el aceite en el depósito desaparece rápidamente, **que** podría indicar que hay fugas u otros problemas en el sistema.

- **Nota:** Si nota algún problema con el sistema de lubricación, informe a su supervisor o líder de equipo de inmediato para que pueda corregirlo.

- **Tenga cuidado con la contaminación.**
- Los sistemas de lubricación tienen filtros, pero no manejan muy bien la contaminación. Los contaminantes típicos incluyen:
- Otro aceite: nunca mezcle los aceites.
- Suciedad y otra basura: utilice siempre latas limpias para transportar el aceite.
