## [Page 2]

# Preparing for a Setup

Paperwork is the first consideration when you prepare to set up a job. It's important for you to be able to find, read, and interpret this paperwork to set up a job successfully. In some cases, you may have to fill in information as you proceed.

The next consideration is your approach to the setup itself. A successful setup is an organized, planned event. We'll give you some basics here, with more details outlined in Module 9.

# Paperwork

The paperwork most common to a setup is the following:

- Production Schedule
- Job Packet
- Quality Paperwork
- Blueprint
- Process Log Records
- Setup Documentation

You may want copies of these to refer to as we discuss them in more detail.

### Production Schedule

A Production Schedule is a listing of the jobs for which we have orders. These jobs are listed in the order in which they must run. One look at the schedule should give you the jobs to be run in each press and the order they must be run in.

This schedule also gives you other valuable information. This includes:

- Raw material size
- Due date
- Press information
- Part number
- Customer name
- Run quantity

See your Supervisor to identify other information on this report that may help you perform your job successfully.

## [Page 3]

### Job Packet

A Job Packet is a bundle of information that you need to set up and run a job. It commonly includes the following items:

- A blueprint of the part being produced.
- A card or list with information about routing steps, run quantities, due date, and other specific job run information.
- Quality reports and forms.
- Setup documentation information.
- Any other information needed to run the job successfully.

This packet normally includes all of the paperwork you'll need to run a job. See your Supervisor for further information about the job packet and where it can be found.

### Quality Paperwork

The quality forms and reports available for each job give you the details of what is a quality part. They list the dimensions to check, how to check them, how many parts to check each time, and how often they must be checked. Typical quality forms include the following:

- First piece inspection.
    - Outlines the procedures for giving a first piece inspection prior to running a job.
- Operator instruction sheet.
    - Dimensions to be checked.
    - Frequency of checks.
    - Sample size.
    - Method of measurement.
- Operator checksheet.
    - A method of verifying that checks have been made on a regular schedule.
- SPC charts.

See your Supervisor for further information and help in finding and using this quality paperwork effectively.

### Blueprint

The Blueprint is your bible for producing a quality part. It includes the specific dimensions for the part.

## [Page 4]

### Process Log Records

The process log is a method of keeping a diary of what happens during a job run. It includes everything from coil changes to detailed information about technical problems. It must include both the reason for the problem and what was done to correct it.

The importance of the process log before setup is that you can acquaint yourself with how the job has run in the past. It gives you good insight into specific problem areas, things to prepare for, and things to watch out for.

See your Supervisor for specific information regarding process log sheets.

### Setup Documentation

Setup documentation includes measurements used to shorten setup times. These measurements could include adjustments, timing, and other settings. We'll deal with the technical issues in the next two modules. See your Supervisor for additional information on finding and using setup documentation.

# Setup Approach

As we mentioned earlier, a successful setup is planned, organized, and precise. A setup includes the following ingredients. If any of these are missing, the setup cannot be viewed as successful:

- Preparing a set of tools and a press for long-term, consistent operation.
- Mounting, aligning, and adjusting the tools to a precise standard.
- Producing a quality part, consistently, throughout the entire run.
- The setup itself will take a minimum time to complete and will assure the shortest possible run-time.
- Setup time begins when a job is run out in a press. This is when the setup "*clock*" starts for the next job going in. The setup clock stops after the press is again placed into production with the next job.

The modules found in this section include methods to achieve each of these ingredients. Keep these things in mind as you proceed through each of the modules.
